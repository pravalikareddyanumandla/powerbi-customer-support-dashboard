# powerbi-customer-support-dashboard
power BI dashboard analyzing customer support requests using snowflake 
This project analyzes customer support request data using Power BI to identify trends, performance metrics, and issue distribution. The dataset is a sample dataset simulating real-world support operations.
Tools Used
Power BI
Snowflake
SQL
DAX
Key Features
KPI tracking (Total Requests, Completed, Open, Completion %)
Monthly trend analysis
Request type distribution
Interactive slicers for filtering
